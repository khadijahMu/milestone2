# Express Authentication API (Milestone 2)

##  Project Description
This is a simple authentication API built using **Node.js, Express, MongoDB, JWT, and bcrypt**.  
It includes **user signup, login, and authentication with JWT tokens**.

---

##  **Technologies Used**
- **Node.js**
- **Express.js**
- **MongoDB (Mongoose)**
- **JWT (JSON Web Token)**
- **bcrypt (Password Hashing)**
- **Render (https://milestone-2-1-7tqg.onrender.com/)**
